# Ecommerce API (Laravel)

Backend REST API for a multi-role ecommerce system built with Laravel.

## Overview
This API serves three Flutter applications:
- Customer App
- Driver App
- Admin App

It handles authentication, business logic, and database operations.

## Features
- User authentication (customers, drivers, admins)
- Role-based authorization
- Product management
- Order management
- Delivery status updates
- RESTful API endpoints

## Tech Stack
- Laravel
- PHP
- MySQL
- REST API
- Sanctum / JWT
- Git

## Setup (Local)
1. Clone the repository
2. Run `composer install`
3. Create `.env` file
4. Run `php artisan migrate`
5. Run `php artisan serve`
